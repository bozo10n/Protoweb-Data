16th November 2024 - November 16th Alpha Updates

We've reached 1000 PRs! Sincerest appreciation to all contributors that are working on this.

Content Changes

Features

  • Digsite Quest has been added
  • More NPC stats and organization

Fixes

  • Typo in Restless Ghost
  • Added 2.5% combat XP scaling for Chronozon
  • Changed Chronozon's death/regen behaviour
  • doubleobjbox proc $scale argument - fixes some stat advancement messages
  • Remove Staff of Armadyl when completing Temple of Ikov
  • Bird Chompy Bird Hunting bug fixes

Engine Changes

Features

  • changestat trigger
  • ai_spawn trigger
  • 0 XP spawnhitting bug implemented
  • NPCs wandering will teleport if stuck for ~5 minutes
  • Pre-2006 walking logic for 2x2 and larger NPCs
  • Automatic image quantization. We use the original packed sprites, so this code only runs when custom images are added to the data folder

Fixes

  • New NPC renderer removes permanent NIDs properly
  • NPC respawn and hunt processing order moved
  • Statically allocate int & string locals in script runtime
  • Stop facing entities after opheld runs
  • Jagfile decompression could fail using standard libbz2 due to a packing oversight. The client was unaffected
  • Removed quotes from constants referenced in scripts
