Enabling Always Run for Id Tech 1 Games (Doom/Heretic/Hexen/Strife)


To do this, you will need to find the DEFAULT.CFG file in the respective game's directory (this is called HERETIC.CFG, HEXEN.CFG or STRIFE.CFG for those respective games). Once there, open it in a notepad program of your choice (Windows 10's notepad should work fine). Scroll down until you find the variable called joyb_speed and change it's value to 29.
