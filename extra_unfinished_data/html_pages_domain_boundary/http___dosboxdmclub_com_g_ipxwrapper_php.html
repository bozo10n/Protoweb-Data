IPXWrapper (DOSBox UDP Encapsulation)

IPXWrapper is a wrapper library which emulates IPX support while tunneling the packets over UDP. This allows you to run legacy games and other software that require IPX without having to install the IPX driver which is unavailable as of Windows Vista/Server 2008.

This updated build IPXWrapper allows for IPX emulation for certain windows games by connecting to an active DOSBox IPXnet server, allowing players to play together like a VPN.

You can grab the latest version: Github.com

1. Extract the contents of the ZIP's inner folder to the desired game's directory. If done correctly, there should be an ipxconfig.exe in the same directory as your game's executable.

2. Run directplay-win64.reg or directplay-win32.reg, depending on your OS architecture.

3. Open ipxconfig.exe and set the Encapsulation type to DOSBox UDP Encapsulation

4. Set the address to ipx.soulsphere.org and the port to 10000 (you can use other addresses/port numbers but we use those specific ones for events)

5. Run the game and host/join via IPX and enjoy!

NOTE : For Windows 10/11 Users, you can find out if your system is 32-bit or 64-bit by going to Settings (from the Start Menu) and going to System > About.
