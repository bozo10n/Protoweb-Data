About Messenger Plus! Downloads

The MsgPlus downloads are in encrypted 7zip (you can use any modern compressed file extractor) files, using the password "escargot" (without quotes).

This is due to an unfortunate false positive from Malwarebytes (and other AVs) blacklisting our sites.

The files are safe to download.
