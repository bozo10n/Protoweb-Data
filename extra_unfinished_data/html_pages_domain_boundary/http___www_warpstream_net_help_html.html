The following machine configurations are recommended:

Windows 95 / 98 / ME - Setup Guide

  • RealPlayer / Windows Media Player recommended
  • CPU: 90 MHz Pentium-class minimum / 166 MHz Pentium-class recommended
  • RAM: 128 MB minimum / 512 MB recommended
  • GPU: PCI graphics card required / AGP graphics card recommended
  • Browser: Internet Explorer 5.5 / 6 recommended

Windows XP / 2000 / Vista / 7 - Setup Guide

  • Flash Player 9+ recommended
  • CPU: 600 MHz minimum / 1 GHz recommended
  • RAM: 128 MB minimum / 512 MB recommended
  • GPU: PCI graphics card required / AGP graphics card recommended
  • Browser: Internet Explorer 6+ recommended

Windows 8 / 8.1 / 10 / 11 - Setup Guide

  • Flash support with Retrozilla browser
  • RealPlayer supported

Classic Mac OS 7.61 - 9.2.2 - Setup Guide

  • RealPlayer recommended
  • Flash player supported
  • CPU: 200+ MHz PowerPC Mac required
  • RAM: 128 MB minimum / 512 MB recommended
  • Browser: Internet Explorer 5.5 / Netscape 7 recommended

Mac OS X (10.1 - 10.3) - Setup Guide

  • Flash support only. Flash Player 9 required
  • CPU: 200+ MHz PowerPC Mac required
  • RAM: 128 MB minimum / 512 MB recommended
  • Browser: Safari recommended

Mac OS X (10.4 - 10.5) - Setup Guide

  • Flash support only. Flash Player 10.1 required
  • CPU: 500+ MHz PowerPC / Intel Mac required
  • RAM: 128 MB minimum / 512 MB recommended
  • Browser: Safari recommended

Nintendo Wii - Setup Guide

  • Flash support only

Troubleshooting

Flash Player Streaming

Known Issues

  • The 2005 player does not allow going into fullscreen.
  • The 2005 and 2006 players do not allow seeking to unloaded parts of the video.

Flash Player Audio / Video lag

Try switching to the Basic Flash Player on the settings page. If this doesn't work, try switching to the 2006 player and lowering the Flash player quality by right-clicking on the video, and under "Quality", select the Low setting.

If you continue having issues, you may want to look at Real Player or Windows Media Player streaming by clicking the appropriate streaming link on the play page.

Using subtitles

Subtitles are available on select videos. Videos with subtitles have a indicator on the user's channel page. To watch with subtitles, switch to the 2008 player. Note that some features of the 2008 player, such as subtitles, only work with Flash Player 9 or higher.

RealPlayer Streaming

Known Issues

  • Seeking does not work on RealPlayer.
  • Proxy setting may not be set up correctly by default (see below for possible fix).
  • Video may stutter, may have a bad image or not appear at all. (see below for possible fix)

RealPlayer doesn't connect

Try manually setting your proxy in RealPlayer. To do so, open RealPlayer in Windows and select Options > Preferences > Proxy. Under HTTP Options, select "Manually configure HTTP proxy". For server enter wayback.protoweb.org and for port enter 7851. Click OK and then restart your computer. After restarting, streams should now work as expected.

RealPlayer video stutters, corrupt image or video does not appear

Try going into performance settings and disable hardware acceleration. Open RealPlayer in Windows and select Options > Preferences > Performance, and under Video card compatibility, first try unchecking "Use optimized video display" and see if that makes a difference. If not, under Playback performance, you can lower the optimization level using the slider toward "Lowest CPU usage". If you have problems with sound, try different settings under Sound card compatibility.

Windows Media Player Streaming

Windows Media Player known issues

  • Seeking does not work on Windows Media Player.
  • Buffering takes longer on Windows Media Player versions 7 and 8.
  • Videos loop twice on Windows Media Player versions 7 and 8.

Unsupported OS / Browsers / Devices

  • Windows 3.1 / 3.11
  • Sony PlayStation Portable (PSP) - Flash player version does not support video streaming
  • Nintendo DS / 3DS
  • Sega Dreamcast
