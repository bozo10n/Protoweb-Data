31st October 2024 - October 31st Alpha Updates

Happy halloween! Why don't you log in and treat yourself to a ::give green_halloween_mask / red / blue :)

Our long-term uptime testing is looking good - the live server was just at 3 weeks of uptime with no faults, we're really trying to close in on never requiring reboots on any schedule.

Content Changes

Features

  • Ogre potion brewing
  • Brimstail cave
  • Object (item) respawn rates
  • Desert heat
  • Updated ::quest and ::help command menus

Fixes

  • Mark Rashiliyia's Tomb as a multi-way zone
  • Unmark outside parts of Barbarian village as multi-way zones
  • Wrong bone key in objbox
  • Missing p_stopaction for trading and dueling
  • Chaos druid, skeleton mage combat scripts, configs added
  • Witch's House line of sight fix
  • Tutorial island rat were always hitting 1s
  • Changed what slot items go into when equipping/unequipping 2h and shields. It was acting like modern OSRS, but it's not supposed to simply swap slots
  • Meat pizza half had wrong mes
  • No longer allows multiple general random events
  • Null check in eating script
  • Reworked hemenster fishing + general delay adjustments
  • Removed all label jumping within procs (language strictness)
  • Removed the inauthentic last_coord command (devs just have to be aware the time a trigger runs gives no guarantees on the coord, due to protection/queues)
  • Tweaked multicannon projectile speed
  • Revised non-standard crossbow bolt ranged strength bonus
  • Normalized scroll colours when opening
  • Removed br tag from Captain Shanks dialogue
  • Fixed typo in Shilo Village dialogue
  • Added Shilo cart loc options
  • Support inverted locked doors
  • Allow KBD safespotting
  • Added max-stack error messages to RC and bank
  • Moved super strength potion unlock message to correct level
  • Dwarf random would still try to follow after getting angry
  • Prevent players from stealing others' hostile random events (need to confirm this)
  • Prevent searching for junk on tutorial island
  • Trades wouldn't always close the other player's window
  • Removed empty slots on trade confirmation window
  • Add authentic Ned dialogue for when he's shipwrecked

Engine Changes

Features

  • Rewritten pathfinder (using Rust WebAssesmbly) for a 2x speed increase over existing optimized wasm
  • npc and obj respawn scaling by world population
  • buffer_full command
  • Rewritten dev file watcher, improved dev workflow
  • Expanded server console logging
  • Stricter compiler syntax

Fixes

  • Stacking unrevealed objects onto a revealed stack
  • Interacting with objects on a reveal tick
  • Moved delay processing to run earlier in tick
  • Simplified possible errors during user's first setup
  • Clear removed entities from EntityList (memory leak over time)
