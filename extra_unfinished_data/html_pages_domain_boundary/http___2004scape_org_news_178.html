10th November 2024 - November 10th Alpha Updates

Last week's memory stability testing was a success, and we're back to a weekly/maybe quicker update schedule!

Give Family Crest a try, and let us know if there's any visual bugs anywhere as a result of some of the recent engine work. There's some big optimizations under the hood - any time we touch synchronization logic it could result in new problems appearing. From our local testing it all seems fine!

Content Changes

Features

  • Family Crest quest
  • NPC stats defined/updated for many NPCs

Fixes

  • Simplified objbox calls
  • Removed cooldown on Charge spell
  • Duel time limit, duel time out, and logout trigger changes
  • Bronze bar superheating fixed
  • Smithing daggers level-up typo addressed
  • Item spawns updated

Engine Changes

Features

  • Optimized player_info and npc_info packets
  • Optimized scaleByPlayerCount

Fixes

  • Hunt config changes (more properties/logic changes)
  • Fixed a movement bug when clicking multiple destinations and a pathing target in quick succession
  • Players now naively path if the target is on top of them (works differently in OSRS)
  • NPC's huntAll checked huntRange from its base npcType instead of itself
  • Don't send UNSET_MAP_FLAG when standing still
  • Soundscape song no longer crashes
  • Authentic Loc category config code in use
  • Authentic Obj tradeable config code in use, items default to tradeable
  • InvType safety in player save
