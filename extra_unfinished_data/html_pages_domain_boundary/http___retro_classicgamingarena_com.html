Welcome to Classic Gaming Arena!

What is this site all about?
In a nutshell, Classic Gaming Arena is a website based portal used to link up with friends, family or just other gamers to play DOS based IPX network and serial games over the Internet via the use of DOSBox and our small client program, the CGA Client. You can also run your own dedicated server using our CGA Dedicated Server software.

If you would like to know more, continue reading the overview in the help section. If you are ready to get started now, create an account, login and start setting up to wreak some old school multiplayer havoc. If you need some games, stop by our downloads section and grab some shareware and demo versions of games such as Doom, Duke Nukem 3D, Quake and Descent.

If you're looking for someone to play with, checkout the DOSBox Deathmatch Club on Discord!
